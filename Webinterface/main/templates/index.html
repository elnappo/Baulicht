Deine Nachricht über Augsburg!

Hast du schon immer das Bedürfnis gehabt deine Gedankengänge ganz Augsburg mitzuteilen? Hast du einen weltbewegenden Gedanken, aber keine Zeitung, um ihn der Welt nahe zu legen? Dann steig jetzt ein ins Baulicht!

Mit dieser Webseite kannst du deine Texte auf einem am Hotelturm angebrachten Morselicht darstellen und ganz Datschiburg empfängt deine Wellen vom Maiskolben!

Sende deine Nachricht

Open Source

Damit ihr auch in eurer Stadt Nachrichten morsen könnt, haben wir unsere Anwendung Open Source gemacht. So könnt ihr unser Projekt einfach kopieren, nachbauen und loslegen!

Github

Da wir keinen eigenen Server zur Verfügung haben, nutzen wir die exzellenten Dienste von Github, die es uns ermöglichen euch unsere Projektquellen anzubieten.

Python, Qt und C++

Baulicht setzt auf die weit verbreiteten Programmiersprachen/Frameworks Python, Django, C++ und Qt.

dBus

Zur Kommunikation zwischen den Komponenten wird dBus verwendet; eine Schnittstelle zur Inter-Prozess-Kommunikation.

REST API

Falls ihr programmieren könnt, haben wir eine REST API gebaut, mit der ihr auf einfache Weise das Baulicht ansteuern und erweitern könnt.

Webinterface

Zur einfachen Bedienung sowohl von zuhause als auch von Mobilgeräten aus bietet euch Baulicht eine Weboberfläche, von der ihr _EURE_ Nachrichten senden könnt.
